Once you have created a subnet within the system, navigate to that subnet.

You will see a DHCP Pools section listed. There is an 'Add DHCP Pool' button where you can define the start/end pool range and any other lease related timers.

This will then block out the pool range from being used within ONA. If you are using ONA to build DHCP configurations you will also need to associate subnets to your DHCP server that is defined in ONA. That is done in the add dhcp server dialog right next to pools on the display subnet screens.
