Thanks, glad you like it so far.

Currently the only method to do this is to add the anycast IP to a host.. then "share" it with other hosts. This sharing is typically ment for things like VRRP, HSRP, CARP etc where you are sharing a normal IP between a few hosts.

While you could track your anycast IPs this way, I'm not sure it is the "right" way to do it in ONA. So far I've not had the need or request to track anycast IPs on any kind of large scale so I've not really thought through the whole problem. You can get to the "share" feature when you hit the green plus icon next to an IP address on a host.

Give that a try and see if it meets your needs.
